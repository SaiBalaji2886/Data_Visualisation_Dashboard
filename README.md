
# DataViz Dashboard — CSV Visualizer

**DataViz Dashboard** is a dynamic web application that allows users to upload CSV files and visualize their data through interactive charts. It is built using **HTML, CSS, and JavaScript**, leveraging **Chart.js** for chart rendering and **PapaParse** for CSV parsing.

## Features

- **Upload CSV Files:** Drag & drop or browse CSV files to visualize data instantly.
- **Interactive Charts:** Generate Line, Bar, and Pie charts dynamically.
- **Column Selection:** Choose X-axis and Y-axis columns from uploaded data.
- **Data Preview:** View your CSV in a table format with search and filter options.
- **Download Options:** Export charts as PNG images and download processed CSV files.
- **Dark Mode Toggle:** Switch between light and dark themes.
- **Responsive Design:** Works seamlessly across different screen sizes.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/SaiBalaji2886/Data_Visualisation_Dashboard.git
````

2. Open `index.html` in your web browser.

No backend setup required; this is a **purely front-end project**.

---

## Usage

1. Open the dashboard in your browser.
2. Upload a CSV file using the welcome section or sidebar.
3. Select columns for **X-axis** and **Y-axis**.
4. Click **Render** to generate charts.
5. Preview your data in the table below.
6. Download charts or the CSV if needed.

## Technologies Used

* **HTML5 & CSS3** — Structure and styling of the dashboard.
* **JavaScript** — Interactivity and logic for chart rendering.
* **Chart.js** — Library for creating interactive charts.
* **PapaParse** — Library for parsing CSV files.
* **Google Fonts (Poppins)** — Typography.
